# Calculator Web App

Веб-приложение для лабораторной работы №2 по дисциплине «Технологии разработки серверных приложений на PHP».

## Описание

Приложение реализует игру `calculator` в виде веб-сайта на PHP с использованием SQLite.

Возможности:
- ввод имени игрока;
- генерация случайного арифметического выражения;
- проверка ответа;
- сохранение результата игры в базу данных SQLite;
- просмотр истории игр.

## Структура

- `public` — корень сайта
- `db` — база данных SQLite
- `src` — вспомогательные модули приложения

## Запуск

Из каталога `Task02` выполнить:

```bash
php init_db.php
php -S localhost:3000 -t public
```

После этого открыть в браузере:

```text
http://localhost:3000
```
